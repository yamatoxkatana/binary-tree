## Compilation and Execution

To compile and run the program, follow these steps:

**Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/BinaryTree.git
   cd BinaryTree
   ```

**Compile the Program**:


```bash
g++ -o binary_tree BinaryTree.cpp
```

**Run the Program**:
bash
```
./binary_tree
```
